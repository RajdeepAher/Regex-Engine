# Custom Regex Engine

## Build Instructions

1. Create the build directory:
    ```bash
    mkdir build
    ```

2. Navigate into the build directory:
    ```bash
    cd build
    ```

3. Run CMake to generate the necessary build files:
    ```bash
    cmake ..
    ```

4. Build the project:
    ```bash
    make
    ```

5. Clean the build directory:
    ```bash
    make clean_
    ```
